    **Rate limiting strategy for Phase 75:**
    - For pre-auth routes (like `/auth/connect`): rate limit by client IP using `axum::extract::ConnectInfo`
    - For authenticated routes (all routes after auth_extractor_middleware): rate limit by validated PID from JWT claims
