    Create `src-tauri/crates/uc-cli/src/commands/search.rs` and add a grouped search command implementation with these exact public items:

    - `pub enum SearchCommands`
    - `pub async fn run(subcommand: SearchCommands, json: bool, verbose: bool) -> i32`

    Add these concrete subcommands to `SearchCommands`:

    ```rust
    Query {
        query: String,
        #[arg(long)]
        operator: Option<String>,
        #[arg(long = "time-preset")]
        time_preset: Option<String>,
        #[arg(long = "from-ms")]
        from_ms: Option<i64>,
        #[arg(long = "to-ms")]
        to_ms: Option<i64>,
        #[arg(long = "type")]
        file_types: Vec<String>,
        #[arg(long = "ext")]
        extensions: Vec<String>,
        #[arg(long, default_value_t = 50)]
        limit: u32,
        #[arg(long, default_value_t = 0)]
        offset: u32,
        #[arg(long)]
        detailed: bool,
    },
    Status,
    ```

    `run()` must:

    - construct `DaemonClientContext::from_env()` for daemon-backed execution
    - route `Query` to `ctx.search_client().query(SearchQueryRequest { ... })`
    - route `Status` to `ctx.search_client().status()`
    - return `EXIT_DAEMON_UNREACHABLE` when daemon connection info is unavailable
    - return `EXIT_ERROR` for daemon request failures or local flag-shape failures

    Local validation rules must be concrete:

    - if `from_ms` is present without `to_ms`, print `Error: --from-ms and --to-ms must be provided together`
    - if `to_ms` is present without `from_ms`, print the same message
    - do **not** reject inline `AND` / `OR` or mixed operators locally

    Wire the grouped command into `src-tauri/crates/uc-cli/src/main.rs` with:

    ```rust
    Search {
        #[command(subcommand)]
        subcommand: commands::search::SearchCommands,
    },
    ```

    Add the exact dispatch arm:

    ```rust
    Commands::Search { subcommand } => commands::search::run(subcommand, cli.json, cli.verbose).await,
    ```

    Add `pub mod search;` to `src-tauri/crates/uc-cli/src/commands/mod.rs`.
