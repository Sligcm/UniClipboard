1. Create `src-tauri/crates/uc-app/src/usecases/unlock_encryption_with_passphrase.rs`:

- Define `UnlockWithPassphraseError` enum (thiserror):
  - `NotInitialized` -- "encryption is not initialized, cannot unlock"
  - `ScopeFailed(String)` -- "key scope resolution failed: {0}"
  - `KeySlotLoadFailed(#[source] EncryptionError)` -- "failed to load keyslot: {0}"
  - `MissingWrappedMasterKey` -- "keyslot has no wrapped master key"
  - `KekDeriveFailed(#[source] EncryptionError)` -- "failed to derive KEK from passphrase: {0}"
  - `UnwrapFailed(#[source] EncryptionError)` -- "failed to unwrap master key (wrong passphrase?): {0}"
  - `SessionSetFailed(#[source] EncryptionError)` -- "failed to set master key in session: {0}"

- Define `pub struct UnlockEncryptionWithPassphrase` holding:
  - `encryption_state: Arc<dyn EncryptionStatePort>`
  - `key_scope: Arc<dyn KeyScopePort>`
  - `key_material: Arc<dyn KeyMaterialPort>`
  - `encryption: Arc<dyn EncryptionPort>`
  - `encryption_session: Arc<dyn EncryptionSessionPort>`

- Implement `from_ports()` constructor (same pattern as AutoUnlockEncryptionSession).

- Implement `pub async fn execute(&self, passphrase: Passphrase) -> Result<(), UnlockWithPassphraseError>`:
  1.  Load encryption state. If `Uninitialized`, return `Err(NotInitialized)`.
  2.  Get key scope via `key_scope.current_scope()`.
  3.  Load keyslot via `key_material.load_keyslot(&scope)`.
  4.  Extract `wrapped_master_key` or return `MissingWrappedMasterKey`.
  5.  Derive KEK from passphrase: `encryption.derive_kek(&passphrase, &keyslot.salt, &keyslot.kdf)`.
  6.  Unwrap master key: `encryption.unwrap_master_key(&kek, &wrapped_master_key.blob)`.
  7.  Set in session: `encryption_session.set_master_key(master_key)`.
  8.  Log success via `tracing::info!("Encryption session unlocked via passphrase")`.

- IMPORTANT: Do NOT derive Debug on `Passphrase` parameters or log the passphrase value.

- Add `#[cfg(test)] mod tests` with tests reusing the same mock pattern as `auto_unlock_encryption_session.rs`:
  - `test_unlock_returns_error_when_uninitialized` -- execute returns `UnlockWithPassphraseError::NotInitialized`
  - `test_unlock_succeeds_with_correct_passphrase` -- execute returns `Ok(())`, session.was_master_key_set() is true
  - `test_unlock_propagates_unwrap_error_on_wrong_passphrase` -- mock encryption fails unwrap, returns `UnwrapFailed`
