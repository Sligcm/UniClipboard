```rust
pub fn router() -> Router<DaemonApiState> {
    Router::new()
        .route(http_route::STORAGE_STATS, get(get_storage_stats_handler))
        .route(http_route::STORAGE_CLEAR_CACHE, post(clear_cache_handler))
}
```

**`get_storage_stats_handler`:**

1. Extract `State(state)`. Get runtime or return internal_error.
2. Call `CoreUseCases::new(runtime.as_ref()).get_storage_stats().execute().await`.
3. For `spool_size_bytes`: compute separately using `tokio::fs::metadata` or the `dir_size` utility. Access `runtime.storage_paths().spool_dir` and compute size. Use the same `dir_size` async function pattern from `uc-app/src/usecases/storage/mod.rs` (which is `pub async fn dir_size(path: &Path) -> Result<u64>`). Import it via `uc_app::usecases::storage::dir_size`.
4. For `blob_count`: call `usecases.get_clipboard_stats().execute().await` (from clipboard stats, which returns `ClipboardStats { total_count, ... }`). Use `total_count` as a proxy for blob count (each entry ~ 1 blob). If the call fails, default to 0.
5. Build `StorageStatsResponse` (daemon-transport DTO, not reusing domain type):
   ```rust
   #[derive(Serialize)]
   struct StorageStatsResponse {
       total_size_bytes: u64,
       blob_count: u64,
       database_size_bytes: u64,
       cache_size_bytes: u64,
       spool_size_bytes: u64,
   }
   ```
   Map:
   - `total_size_bytes` = `stats.total_bytes` + `spool_bytes`
   - `blob_count` = `clipboard_stats.total_count as u64` (or 0 on error)
   - `database_size_bytes` = `stats.database_bytes`
   - `cache_size_bytes` = `stats.cache_bytes`
   - `spool_size_bytes` = computed from spool_dir
6. Return `Json(json!({ "data": response, "ts": chrono::Utc::now().timestamp_millis() }))`.

**`clear_cache_handler`:**

1. Define request DTO:

   ```rust
   #[derive(Deserialize)]
   struct ClearCacheRequest {
       #[serde(default)]
       confirmed: bool,
   }
   ```

2. Extract `State(state)` and body using `Result<Json<ClearCacheRequest>, JsonRejection>`. If body parse fails OR `confirmed` is false, return:

   ```rust
   (StatusCode::BAD_REQUEST, Json(json!({
       "code": "confirmation_required",
       "message": "set confirmed: true to proceed with cache clear"
   })))
   ```

3. Get runtime, create CoreUseCases, call `usecases.clear_cache().execute().await`.
4. On success, return `Json(json!({ "success": true, "freed_bytes": freed }))`.
5. On error, return `internal_error(e)`.
