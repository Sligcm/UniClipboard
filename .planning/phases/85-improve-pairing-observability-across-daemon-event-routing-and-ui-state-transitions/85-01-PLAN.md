<objective>
Establish the backend observability contract for Phase 85. Make daemon emission and bridge translation diagnosable for one pairing session without changing pairing behavior.

This wave defines the minimum shared metadata and structured logs needed to answer:

1. what daemon event was emitted
2. how the websocket bridge translated it
3. whether the routing decision was expected
   </objective>
