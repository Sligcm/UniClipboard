Create the search foundation Phase 92 actually needs before any daemon route work starts: evolve the search query contract to carry pagination truth, add stable search HTTP/WS constants, wire a single search bundle into `AppDeps`, expose search accessors from `CoreUseCases`, and update the existing SQLite adapter to return authoritative `total` / `has_more` metadata. This plan prevents shallow route-layer workarounds and gives later daemon plans a clean runtime surface.
</objective>
