## 0.10.0-alpha.4 - 2026-05-16

### Features

- Redesign the Devices page with clearer overview cards, separate tabs for paired devices and mobile sync, and centered settings dialogs
- Show each clipboard item's source and per-device sync status in the details view, using device names when available

### Fixes

- Keep the macOS window controls aligned with the custom title bar
- Allow text in clipboard previews to be selected and copied
- Show a clearer error when copying a file entry whose original source file was moved or deleted
- Prevent tooltips from reopening by themselves after returning to the app on Windows
- Remove shortcut options from Settings that looked editable but did not actually work
