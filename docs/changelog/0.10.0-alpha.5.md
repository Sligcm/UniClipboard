## 0.10.0-alpha.5 - 2026-05-16

### Features

- Redesign the Devices page with clearer device and mobile sync tabs, card grids, and centered settings dialogs.
- Show where each clipboard item came from and whether it has synced to paired devices, with live updates in the detail view and quick panel.
- Support previewing and syncing large image files copied from Finder without showing Finder thumbnails.

### Fixes

- Allow selecting text inside clipboard previews.
- Show a clear message when a saved file clipboard entry points to a file that was moved or deleted.
- Remove shortcut settings that looked configurable but had no effect.
- Keep macOS window controls aligned with the custom title bar.
- Close the update dialog immediately after choosing to download an update in the background.
- Prevent tooltips from reopening by themselves after returning to the app on Windows.
