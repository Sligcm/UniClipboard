## 0.10.0-alpha.5 - 2026-05-16

### Features

- 重新设计设备页，提供更清晰的设备与手机同步标签页、卡片网格和居中的设置弹窗。
- 在详情视图和快捷面板中显示每条剪贴板内容的来源与配对设备同步状态，并支持实时更新。
- 支持预览和同步从 Finder 复制的大尺寸图片文件，并避免显示 Finder 缩略图。

### Fixes

- 允许在剪贴板预览中选择文本。
- 当已保存的文件类剪贴板内容指向已移动或删除的文件时，显示清晰提示。
- 移除看起来可配置但实际无效的快捷键设置。
- 让 macOS 窗口控制按钮与自定义标题栏保持对齐。
- 选择在后台下载更新后，立即关闭更新弹窗。
- 修复 Windows 上切回应用后工具提示自动重新显示的问题。
