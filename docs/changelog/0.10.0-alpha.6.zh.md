## 0.10.0-alpha.6 - 2026-05-16

### Features

- 为 Linux 和 macOS 增加一行命令安装与卸载方式。
- 重新设计设备页面，提供更清晰的配对设备和移动同步标签页、卡片网格以及居中的设置对话框。
- 在剪贴板条目中显示来源和配对设备同步状态，并在详情视图和快速面板中实时更新。
- 支持预览和同步从 Finder 复制的大型图片文件，不再显示 Finder 缩略图。

### Fixes

- 允许选中剪贴板预览中的文本。
- 当已保存的文件剪贴板条目指向已移动或删除的源文件时，显示更清楚的提示。
- 移除看起来可配置但实际不生效的快捷键设置。
- 保持 macOS 窗口控制按钮与自定义标题栏对齐。
- 选择后台下载更新后，立即关闭更新对话框。
- 防止 Windows 上返回应用后工具提示自动重新弹出。
