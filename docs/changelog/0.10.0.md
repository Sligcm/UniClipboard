## 0.10.0 - 2026-05-17

### Features

- Add a reset-and-start-over option on the unlock screen for users who forgot their passphrase (#690)
- Add background update downloads with clearer sidebar states for available, downloading, and ready-to-install updates (#691)
- Turn background update downloads on by default so supported installers can be ready after an update check (#771)
- Redesign the Devices page with clearer paired-device and mobile-sync tabs, card grids, and centered settings dialogs (#726)
- Show where each clipboard item came from and whether it has synced to paired devices, with live updates in the detail view and quick panel (#732, #736, #746, #749)
- Support previewing and syncing large image files copied from Finder without showing Finder thumbnails (#748)
- Add one-line install and uninstall commands for Linux and macOS (#750)
- Show package-manager upgrade instructions for users who installed the Linux deb or rpm package (#752)
- Add a Quick Panel setting so the panel can be turned on or off immediately (#764)
- Add one-click copying for the current mobile sync address in the mobile sync settings dialog (#775)
- Add a button in General settings to open the app log folder from the file manager (#778)

### Fixes

- Improve Windows quick panel paste reliability in apps where the previous input field did not regain focus correctly (#692)
- Make the main window feel more native by preventing accidental selection of interface text and hiding the browser-style right-click menu while keeping clipboard content selectable (#700)
- Improve pairing reliability when the same device is reachable in multiple ways, preventing later pairing attempts from hanging or timing out after an initial success (#696)
- Prevent tooltips from reopening by themselves after returning to the app on Windows (#707)
- Allow text in clipboard previews to be selected and copied (#727)
- Show a clearer error when copying a file entry whose original source file was moved or deleted (#729)
- Remove shortcut settings that looked configurable but had no effect (#730)
- Keep the macOS window controls aligned with the custom title bar (#738)
- Close the update dialog immediately after choosing to download an update in the background (#743)
- Improve the visibility of the update button and device status colors across themes (#766)
- Prevent a crash that could happen when synced file history pointed to cache files that were already missing (#767)
- Prevent a crash that could happen when receiving an image from another device (#772)
- Fix Snap startup issues that could show a localhost connection error or fail when the system keyring is unavailable (#774)
- Keep the clipboard selection on the newest item when it becomes first in the list (#777)
- Show all available mobile sync addresses when Bind IP is set to Auto, so phone setup no longer depends on guessing the right network address (#783)
