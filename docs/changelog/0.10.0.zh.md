## 0.10.0 - 2026-05-17

### Features

- 在解锁页面新增“重置并重新开始”入口，方便忘记口令的用户重新设置本机 (#690)
- 新增后台下载更新能力，并在侧边栏中更清楚地区分可更新、下载中和可安装状态 (#691)
- 默认开启后台下载更新，检查到新版本后支持的安装包会提前准备好 (#771)
- 重新设计设备页面，提供更清晰的配对设备和移动同步标签页、卡片网格以及居中的设置对话框 (#726)
- 在剪贴板条目中显示来源和配对设备同步状态，并在详情视图和快捷面板中实时更新 (#732, #736, #746, #749)
- 支持预览和同步从 Finder 复制的大型图片文件，不再显示 Finder 缩略图 (#748)
- 为 Linux 和 macOS 增加一行命令安装与卸载方式 (#750)
- 对通过 Linux deb 或 rpm 包安装的用户，显示系统包管理器升级指引 (#752)
- 新增快捷面板开关，可立即启用或停用快捷面板 (#764)
- 在移动端同步设置弹窗中新增当前同步地址的一键复制功能 (#775)
- 在通用设置中新增按钮，可直接从文件管理器打开应用日志文件夹 (#778)

### Fixes

- 提升 Windows 快捷面板在部分应用中的自动粘贴可靠性，修复返回原输入框后焦点未正确恢复的问题 (#692)
- 主窗口更贴近原生应用体验，避免误选界面文字并隐藏浏览器式右键菜单，同时保留剪贴板内容可选中 (#700)
- 提升同一设备可通过多种方式连接时的配对可靠性，避免首次成功后后续配对卡住或超时 (#696)
- 修复 Windows 上切回应用后工具提示可能自动重新弹出的问题 (#707)
- 允许选中并复制剪贴板预览中的文本 (#727)
- 当文件条目的原始源文件被移动或删除时，显示更清楚的复制失败提示 (#729)
- 移除看起来可配置但实际不生效的快捷键设置 (#730)
- 保持 macOS 窗口控制按钮与自定义标题栏对齐 (#738)
- 选择后台下载更新后，立即关闭更新对话框 (#743)
- 改进不同主题下更新按钮和设备状态颜色的可见性 (#766)
- 修复同步的文件历史指向已丢失缓存文件时可能导致应用崩溃的问题 (#767)
- 修复从其他设备接收图片时应用可能崩溃的问题 (#772)
- 修复 Snap 启动时可能显示 localhost 连接错误，或在系统钥匙串不可用时启动失败的问题 (#774)
- 当最新剪贴板内容出现在列表首位时，保持当前选择跟随到该内容 (#777)
- 当绑定 IP 设置为自动时，直接显示所有可用的移动同步地址，手机设置时不再需要猜测正确的网络地址 (#783)
