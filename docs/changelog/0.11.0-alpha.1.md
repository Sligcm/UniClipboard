## 0.11.0-alpha.1 - 2026-05-20

### Features

- Improve the Linux installer so it can choose working Snap or COPR install paths on supported distributions, while still allowing manual format selection (#793)

### Fixes

- Send plain text to mobile clients when copied content also has rich text, so text no longer appears as raw formatting markup (#808)
- Show mobile device registration and settings feedback inside the dialog, including field errors, invalid ports, copy failures, and saved settings confirmation (#792)
- Separate discarding a newly registered mobile device from completing setup after saving credentials (#788)
- Keep clipboard delivery status updates responsive when one paired device is slow or unreachable (#787)
- Refresh deferred clipboard delivery status one device at a time as each device finishes (#790)
- Show a visible receiving placeholder for common screenshot and image syncs (#786)
