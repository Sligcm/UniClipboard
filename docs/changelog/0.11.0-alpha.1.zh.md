## 0.11.0-alpha.1 - 2026-05-20

### Features

- 改进 Linux 安装脚本，使其可在受支持的发行版上自动选择可用的 Snap 或 COPR 安装路径，同时仍支持手动指定安装格式 (#793)

### Fixes

- 当复制内容同时包含富文本和纯文本时，移动端会收到纯文本，避免文本显示成原始格式标记 (#808)
- 移动设备注册和设置反馈会显示在弹窗内，包括字段错误、无效端口、复制失败和设置已生效提示 (#792)
- 新注册移动设备的凭据弹窗现在区分放弃注册和保存凭据后完成设置 (#788)
- 当某个已配对设备较慢或不可达时，剪贴板投递状态更新不再被长时间阻塞 (#787)
- 后台延迟完成的剪贴板投递状态会随每台设备完成而逐个刷新 (#790)
- 常见截图和图片同步时会显示可见的接收中占位提示 (#786)
