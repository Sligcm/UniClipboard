## 0.11.0-alpha.6 - 2026-05-23

### Features

- Update paired device online status faster when a device comes back online, while reducing background traffic from the Devices page (#831)
- Refresh the desktop app icon with new artwork (#832)
- Add a cancel action for incoming file transfers and new CLI commands to send and receive files (#833)

### Fixes

- Make large file transfers more reliable by stopping timed-out or cancelled receives, showing cancellations separately from failures, keeping sender progress accurate until the real file finishes, and preventing progress updates from slowing down the app (#833)
