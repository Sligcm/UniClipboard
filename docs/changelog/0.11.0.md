## 0.11.0 - 2026-05-24

### Features

- Add a Resend action for local clipboard history items so users can retry delivery to devices that missed an item (#806)
- Add custom relay settings with a reachability test for users who run their own relay nodes (#674)
- Add background update checks with desktop notifications for new releases and auto-download when enabled (#823)
- Improve mobile device setup with a scan-to-add QR code that auto-fills connection details and a separate Shortcut install flow (#811)
- Let the Linux quick installer choose better install paths for more distributions, including Snap on Ubuntu 20.04 and COPR on Fedora/RHEL/openSUSE (#793)
- Add cancellable file transfers, clearer cancelled states, and CLI commands for sending or receiving files (#833)
- Refresh the desktop app icon (#832)

### Fixes

- Prevent copied files from Finder from sometimes missing clipboard history (#821)
- Send plain text to mobile clients when a clipboard item also contains rich text, instead of exposing raw formatting markup (#808)
- Show mobile sync registration and settings errors inside their dialogs so users can fix problems without closing the modal (#792)
- Let users discard a newly added mobile device from the credentials dialog without confirming that credentials were saved (#788)
- Prevent startup crashes on Linux systems where the system tray dependency is missing or broken (#817)
- Keep large file transfer status accurate until the real receive finishes and reduce progress-update slowdowns during big transfers (#833)
