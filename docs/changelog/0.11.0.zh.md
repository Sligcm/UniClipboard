## 0.11.0 - 2026-05-24

### Features

- 为本机剪贴板历史增加“重发”操作，用户可以把未送达的内容重新发送给设备 (#806)
- 增加自定义中继设置和连通性测试，方便使用自建中继节点的用户确认配置可用 (#674)
- 增加后台更新检查，新版本可用时发送桌面通知，并在启用后自动下载 (#823)
- 改进移动设备设置流程，新增可自动填入连接信息的扫码添加二维码，并单独提供快捷指令安装流程 (#811)
- 改进 Linux 一键安装脚本，让更多发行版自动选择合适安装路径，包括 Ubuntu 20.04 使用 Snap、Fedora/RHEL/openSUSE 使用 COPR (#793)
- 增加可取消的文件传输、独立的已取消状态，以及用于发送或接收文件的 CLI 命令 (#833)
- 更新桌面应用图标 (#832)

### Fixes

- 修复从 Finder 复制文件时偶尔不会出现在剪贴板历史中的问题 (#821)
- 修复移动端同步在剪贴板同时包含纯文本和富文本时可能收到原始格式标记的问题 (#808)
- 修复移动同步注册和设置错误在弹窗内不可见的问题，用户现在无需关闭弹窗即可看到并修正错误 (#792)
- 修复移动设备凭据弹窗的放弃流程，用户可以丢弃刚添加的设备，而不必误确认已保存凭据 (#788)
- 修复部分 Linux 系统缺少或损坏系统托盘依赖时应用启动崩溃的问题 (#817)
- 修复大文件传输状态可能过早显示完成的问题，并降低大文件传输进度更新造成的卡顿 (#833)
