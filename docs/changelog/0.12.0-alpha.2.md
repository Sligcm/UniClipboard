## 0.12.0-alpha.2 - 2026-05-25

### Features

- Show available updates in a dedicated in-app window with release notes and install actions instead of relying on a system notification (#854)
- Add a download step to mobile device pairing so first-time phone users can scan the app download page before pairing (#855)
- Show a native macOS receive progress HUD for incoming files with per-transfer cancel controls (#848)

### Fixes

- Prevent clipboard history from appearing empty after an interrupted space switch is resumed (#857)
- Make LAN syncing faster and prevent offline devices from delaying clipboard copies (#852)
- Ignore empty placeholder files from other clipboard tools so they no longer trigger repeated remote sync notifications (#849)
