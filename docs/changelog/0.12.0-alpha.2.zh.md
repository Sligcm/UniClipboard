## 0.12.0-alpha.2 - 2026-05-25

### Features

- 发现新版本时改为打开带有发布说明和安装操作的应用内更新窗口，不再依赖系统通知提示 (#854)
- 移动设备配对新增下载步骤，首次使用手机端的用户可以先扫码打开应用下载页再继续配对 (#855)
- macOS 上接收文件时新增原生进度浮窗，并可直接取消单个传输 (#848)

### Fixes

- 修复空间切换中断后恢复时剪贴板历史可能显示为空的问题 (#857)
- 提升局域网同步速度，并避免已离线设备拖慢剪贴板复制 (#852)
- 忽略其他剪贴板工具写入的空占位文件，避免反复触发远端同步通知 (#849)
