## 0.12.0 - 2026-05-26

### Features

- Show a dedicated in-app update window for new releases, with release notes and install actions that are easier to notice (#854)
- Add a two-step mobile pairing flow that first helps users open the mobile app download page, then shows the pairing QR code (#855)
- Show incoming file transfers on macOS in a floating progress panel with a cancel button (#848)

### Fixes

- Open the update window immediately when a new version is found after returning to the main window, and make that window follow the user's theme (#865)
- Preserve cross-platform clipboard content more reliably, including rich text copied on Windows, non-English URLs copied on Linux, and UTF-8 text synced to macOS (#862)
- Avoid repeated waits when copying while a paired device is offline (#863)
- Recover interrupted space changes during unlock so users do not reopen the app to missing history or an unpaired device (#857)
- Improve same-network file transfer stability and reduce copy delays when a paired device silently goes offline (#852)
- Ignore empty placeholder files from other clipboard tools so they no longer trigger useless remote file-sync notifications (#849)
