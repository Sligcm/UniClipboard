## 0.12.0 - 2026-05-26

### Features

- 新版本可用时会显示独立的应用内更新窗口，更醒目地展示更新说明和安装操作 (#854)
- 移动端配对改为两步流程，先引导用户打开移动 App 下载页，再显示配对二维码 (#855)
- macOS 接收文件时会显示带取消按钮的悬浮进度面板 (#848)

### Fixes

- 回到主窗口后若检测到新版本，会立即打开更新窗口，并让该窗口跟随用户的主题设置 (#865)
- 提升跨平台剪贴板内容保真度，包括 Windows 富文本、Linux 非英文 URL，以及同步到 macOS 的 UTF-8 文本 (#862)
- 复制内容时如果配对设备离线，不再反复等待失败的重连尝试 (#863)
- 解锁时会恢复中断的空间变更，避免重新打开应用后出现历史记录缺失或设备未完成配对 (#857)
- 提升同一网络内文件传输的稳定性，并减少配对设备静默离线时造成的复制卡顿 (#852)
- 忽略其他剪贴板工具产生的空占位文件，避免触发无意义的远端文件同步通知 (#849)
