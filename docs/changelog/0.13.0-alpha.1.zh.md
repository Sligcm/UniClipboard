## 0.13.0-alpha.1 - 2026-05-28

### Features

- 刷新移动设备设置与管理体验，改为优先扫码的二维码流程，提供 App 安装入口、可选择的连接地址（包括 Tailscale 地址），并用统一对话框管理设备详情、密码重置和撤销操作 (#883)
- 在安装、快速开始、配对和移动同步指南中补充 Android 客户端与 APK 下载入口 (#879)
- 新增通过 Tailscale 在桌面端和手机不处于同一局域网时进行移动同步的指引 (#885)
- 新增 FAQ 说明为什么 iOS 不能像桌面端一样静默进行后台剪贴板同步 (#878)

### Fixes

- 修复 Linux 安全存储失败时可能导致应用崩溃，或在系统密钥环不可用时误显示为权限问题的情况 (#882)
- 进一步减少配对设备离线时连续复制内容造成的等待 (#889)
