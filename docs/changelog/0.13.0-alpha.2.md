## 0.13.0-alpha.2 - 2026-05-30

### Breaking Changes

- The mobile sync CLI now uses top-level `add` and `revoke` commands plus a `network` group, replacing the previous `devices`, `settings`, and `lan` groups (#914)

### Features

- **Headless server mode** — run UniClipboard on a VPS or container with no desktop session, with optional mobile sync support (#903, #904)
- Use a stable public address and full HTTPS URLs for more reliable direct connections, QR codes, and install links on self-hosted nodes (#905, #906)
- Add tray menu actions for checking updates and restarting the app without opening the main window (#893)
- Let nearby devices complete first pairing on the same local network even when the internet pairing service is unavailable (#896)

### Documentation

- Add a ready-to-use VPS deployment path with English and Chinese setup guides for self-hosted server nodes (#910, #911)

### Fixes

- Improve Windows clipboard capture so rich text with Chinese, Japanese, or Korean characters no longer crashes the app and screenshots from common apps are captured faster (#894)
- Show expired pairing codes as expired instead of suggesting that the code may have been typed incorrectly (#896)
