## 0.13.0-alpha.2 - 2026-05-30

### Breaking Changes

- 移动端同步 CLI 现在使用顶层 `add`、`revoke` 命令和 `network` 分组，取代之前的 `devices`、`settings`、`lan` 分组 (#914)

### Features

- **无头服务器模式** —— 可在没有桌面会话的 VPS 或容器中运行 UniClipboard，并可选择开启移动端同步支持 (#903, #904)
- 自托管节点可以使用稳定的公网地址和完整的 HTTPS 链接，以提升直连、二维码和安装链接的可靠性 (#905, #906)
- 在托盘菜单中新增检查更新和重启入口，无需打开主窗口即可使用 (#893)
- 同一局域网内的设备现在即使无法访问互联网配对服务，也可以完成首次配对 (#896)

### Documentation

- 新增可直接使用的 VPS 部署方案，并配套自托管服务器节点的中英文部署指南 (#910, #911)

### Fixes

- 改进 Windows 剪贴板捕获，复制包含中日韩字符的富文本不再导致应用崩溃，常见应用中的截图捕获也更快 (#894)
- 配对码过期时会明确提示已过期，而不是提示可能输错了配对码 (#896)
