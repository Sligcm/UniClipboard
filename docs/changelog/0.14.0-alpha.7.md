## 0.14.0-alpha.7 - 2026-06-08

### Fixes

- Show an actionable error screen instead of a blank loading screen when the app cannot connect to the background service (#973)
- Recover startup when an incompatible background service is running but its process record is missing (#980)
- Stop the old background service before installing an update so the installer can replace it cleanly, especially on Windows (#977)
- Use a monochrome menu bar icon on macOS so the tray icon matches light and dark menu bar appearances (#974)
- Fix mobile sync device removal and password rotation so they no longer fail with a not-found error (#972)
