## 0.14.1 - 2026-06-10

### Fixes

- Fix the Windows portable build failing to start because its zip was missing the `uniclipd` background service (#1012)
- Stop the background service from flashing console windows on Windows while installing updates (#1012)
- Stop the upgrade notice from being re-detected on every launch (#1012)
- Open manual updates, release-note links, and download buttons in the system browser and point them at the UniClipboard download page, including Windows portable builds that previously attempted an in-app install or did nothing
- Fix Windows update installs failing on non-English-language systems after the background service is stopped
- Record update download and install outcomes in local logs so failing auto-updates can be diagnosed even with telemetry off (#1012)
- Strengthen local connection security so another process on the same computer cannot infer the background service token from repeated failed connection attempts
