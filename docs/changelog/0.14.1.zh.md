## 0.14.1 - 2026-06-10

### Fixes

- 修复 Windows 便携版因 zip 缺少后台服务 `uniclipd` 而完全无法启动的问题 (#1012)
- 修复 Windows 上安装更新时后台服务闪出控制台窗口的问题 (#1012)
- 修复每次启动都重复检测到升级提示的问题 (#1012)
- 更新窗口中的手动更新、发布说明链接和下载按钮现在统一在系统浏览器中打开并指向 UniClipboard 下载页，Windows 便携版也走此入口，不再尝试应用内安装或无响应
- 修复 Windows 在非英文系统上停止后台服务后安装更新失败的问题
- 更新的下载与安装结果现在会记录到本地日志，即使关闭遥测也能排查自动更新失败的原因 (#1012)
- 加强本机连接安全性，避免同一台电脑上的其他进程通过反复失败的连接尝试推断后台服务令牌
