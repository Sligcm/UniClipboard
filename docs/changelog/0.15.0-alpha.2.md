## 0.15.0-alpha.2 - 2026-06-11

### Features

- Add `uniclip get` so the CLI can immediately read already-synced clipboard history, including text, links, images, and files (#1032)
- Improve mobile pairing QR codes so one code can include all reachable desktop addresses and work across LAN, configured public entry points, and Tailscale-style networks (#1026)

### Fixes

- Prevent mobile sync from offering container bridge addresses that phones cannot reach (#1036)
- Allow a device to be paired again after it was unpaired on only one side, without needing to rebuild the whole space (#1033)
- Fix npm CLI installation by using the scoped `@uniclipboard/cli` package and correctly handling Windows binaries in the package archive (#1031)
