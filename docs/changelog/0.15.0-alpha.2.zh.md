## 0.15.0-alpha.2 - 2026-06-11

### Features

- 新增 `uniclip get`，命令行可以立即读取已经同步到历史中的剪贴内容，包括文本、链接、图片和文件 (#1032)
- 改进移动端配对二维码，一个二维码可以同时包含桌面端的全部可达地址，适用于局域网、已配置的公网入口和 Tailscale 类网络 (#1026)

### Fixes

- 移动端同步不再提供手机无法访问的容器网桥地址 (#1036)
- 设备在仅单侧解除配对后，可以重新配对，不再需要重建整个空间 (#1033)
- 修复 npm 命令行安装：改用 scoped 包 `@uniclipboard/cli`，并正确处理 Windows 包中的可执行文件 (#1031)
