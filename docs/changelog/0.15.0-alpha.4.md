## 0.15.0-alpha.4 - 2026-06-12

### Fixes

- Let the daemon start normally on Linux servers and containers without a graphical session by automatically disabling system clipboard access there (#1055)
- Make Linux clipboard capture more reliable when X11 or XWayland apps update the clipboard while the app is already reading it (#1054)
- Improve file clipboard entries by showing the deleted-file message, preserving cancelled transfer status after reload, and hiding unknown file sizes instead of showing bogus values (#1053)
