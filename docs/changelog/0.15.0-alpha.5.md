## 0.15.0-alpha.5 - 2026-06-13

### Features

- You can now edit a mobile sync device's label, username, and password from its device dialog, with updated one-time credentials shown after changes (#1065)

### Fixes

- Long device and mobile sync settings dialogs now stay within the window and scroll without hiding their header or footer (#1067)
- Fixed Linux text clipboard sync so text restored from another device can be pasted into apps such as Firefox and GTK apps (#1061)
- Fixed startup recovery after updates so the app can replace a stuck older background service and Retry restarts it instead of requiring a manual process kill (#1063)
- Fixed blank or crashing windows on Linux Wayland sessions such as Hyprland and Sway (#1062)
- Fixed Linux startup hangs when no unlocked system keyring service is available by falling back instead of waiting indefinitely (#1058)
