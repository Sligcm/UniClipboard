## 0.15.0-alpha.5 - 2026-06-13

### Features

- 现在可以在移动同步设备对话框中编辑设备名称、用户名和密码，修改后会显示一次性的新连接凭据 (#1065)

### Fixes

- 较长的设备设置和移动同步设置对话框现在会保持在窗口内滚动，不再遮住顶部或底部操作区 (#1067)
- 修复 Linux 文本剪贴板同步问题，从其他设备恢复的文本现在可以粘贴到 Firefox 和 GTK 应用等程序中 (#1061)
- 修复更新后的启动恢复流程，应用现在可以替换卡住的旧后台服务，Retry 也会重启后台服务，不再需要手动结束进程 (#1063)
- 修复 Linux Wayland 会话（如 Hyprland 和 Sway）中窗口空白或崩溃的问题 (#1062)
- 修复 Linux 上没有可用且已解锁的系统钥匙串服务时启动卡住的问题，现在会自动降级而不是无限等待 (#1058)
