## 0.15.0 - 2026-06-14

### Features

- 新增移动同步设备编辑流程，可修改设备标签、用户名或密码，并在需要时扫码更新后的凭据 (#1065)
- 移动同步二维码现在会包含桌面端所有可访问地址，让手机在局域网、Tailscale 或已配置公网入口下更容易连上 (#1026)
- 新增 `uniclip get`，可通过 CLI 直接读取已经同步到历史记录里的剪贴板内容，无需等待下一次传入 (#1032)
- CLI 现在可通过 npm 安装，方便终端和脚本场景使用 (#1028)

### Fixes

- 较长的对话框现在会滚动正文，同时保持标题栏和底部按钮可见 (#1067)
- 修复 Linux 上已同步文本无法粘贴到 Firefox、GTK 等应用的问题，并避免收到剪贴板更新后粘贴卡顿 (#1061)
- 修复更新或重启后旧后台服务仍在退出时可能导致应用卡住的问题 (#1038, #1063)
- 避免 Linux 会话中系统密钥环不可用或未解锁时导致启动卡住 (#1058)
- 改进 Linux X11 剪贴板捕获，快速切换或响应较慢的剪贴板来源不再容易漏同步内容 (#1054)
- 无头主机现在可以在没有系统剪贴板的情况下启动，而不是在后台服务启动时失败 (#1055)
- 移动同步地址选择器不再显示容器虚拟网卡 (#1036)
- 当旧的过期设备记录仍存在时，设备现在可以重新配对成功 (#1033)
- 避免 Wayland 下 WebKit DMABUF 渲染器导致崩溃或窗口空白 (#1062)
- 修复 Windows npm CLI 包对嵌套路径的处理 (#1031)
