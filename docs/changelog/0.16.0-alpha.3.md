## 0.16.0-alpha.3 - 2026-06-17

### Breaking Changes

- Replace `uniclip switch-space` with `uniclip join --switch` for moving an already set up device to another space (#1095)

### Features

- Simplify CLI commands so search queries can run as `uniclip search <query>`, member listing can optionally refresh online state, and `uniclip upgrade` shows status by default (#1095)

### Fixes

- Make Linux image syncing prefer pasteable image formats when apps offer multiple image variants (#1089)
- Improve Linux X11 clipboard capture so text copied from apps that publish clipboard data slowly is less likely to be missed (#1096)
- Keep `uniclip get` text output readable in an interactive terminal without changing piped or redirected output (#1091)
