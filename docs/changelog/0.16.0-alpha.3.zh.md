## 0.16.0-alpha.3 - 2026-06-17

### Breaking Changes

- 将已设置设备切换到另一个空间的 CLI 入口从 `uniclip switch-space` 改为 `uniclip join --switch` (#1095)

### Features

- 简化 CLI 命令：搜索可直接使用 `uniclip search <query>`，成员列表可按需刷新在线状态，`uniclip upgrade` 不带子命令时默认显示状态 (#1095)

### Fixes

- Linux 上当应用提供多个图片版本时，优先同步可正常粘贴的图片格式 (#1089)
- 改善 Linux X11 剪贴板捕获，降低从较慢发布剪贴板数据的应用复制文本时漏同步的概率 (#1096)
- 让 `uniclip get` 的文本输出在交互式终端中更易读，同时不改变管道或重定向输出 (#1091)
