## 0.4.0-alpha.2 - 2026-04-02

### Features

- Improve setup and pairing flow so failed steps can be retried more smoothly.
- Add more reliable media loading paths for clipboard previews and file content.
- Improve real-time connection stability by detecting stale sessions and recovering faster.
- Add support for browser-based real-time sessions in web deployments.

### Fixes

- Fix a black screen that could appear after unlocking.
- Respect the Auto Unlock preference when the app starts.
- Improve startup reliability to reduce cases where the app waits too long to become ready.
- Fix clipboard sync edge cases that could cause duplicate or out-of-order updates.
- Fix settings sync mismatches that could cause inconsistent device configuration.
