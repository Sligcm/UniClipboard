## 0.4.0-alpha.3 - 2026-04-06

### Features

- Improve setup and pairing flow with clearer waiting guidance, in-app status feedback, and a reset option for transient session state without losing completed progress.
- Improve clipboard preview and media loading reliability for images and files across desktop and web scenarios.
- Improve real-time connection resilience by detecting stale sessions sooner and recovering faster, with browser-based real-time sessions supported in web deployments.
- Improve CLI setup, pair, connect, and reset commands so local daemon lifecycle is handled automatically for more predictable runs.

### Fixes

- Fix a black screen that could appear after unlocking.
- Respect the Auto Unlock preference when the app starts.
- Improve startup readiness and packaging reliability to reduce cases where the app waits too long to become ready.
- Fix clipboard sync edge cases that could cause duplicate, out-of-order, or repeated restore updates.
- Fix settings sync mismatches that could cause inconsistent device configuration.
- Fix quick panel state consistency after daemon restarts, including lock status and theme synchronization across windows.
- Restore macOS file previews.
- Fix setup and pairing edge cases that could interrupt active sessions or mis-handle passphrase verification steps.
