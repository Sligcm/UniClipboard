## 0.4.0-alpha.3 - 2026-04-06

### Features

- 改进初始化与配对流程，提供更清晰的等待提示、应用内状态反馈，以及可重置临时会话状态且不丢失已完成进度的能力。
- 提升剪贴板图片与文件预览在桌面端和 Web 场景下的加载稳定性。
- 提升实时连接韧性，可更快识别失效会话并恢复，并在 Web 部署场景下支持基于浏览器的实时会话。
- 改进 CLI 的 setup、pair、connect、reset 命令，自动处理本地守护进程生命周期，执行过程更稳定可预期。

### Fixes

- 修复解锁后可能出现黑屏的问题。
- 修复应用启动时未正确遵循自动解锁设置的问题。
- 提升启动就绪与打包可靠性，减少应用长时间未就绪的情况。
- 修复剪贴板同步边界场景，减少重复、顺序异常或反复恢复的问题。
- 修复设置同步不一致导致的设备配置异常。
- 修复守护进程重启后快速面板状态不一致的问题，包括锁定状态和多窗口主题同步。
- 修复 macOS 文件预览不可用的问题。
- 修复初始化与配对中的边界问题，避免活跃会话被意外中断或口令校验流程处理异常。
