## 0.4.0-alpha.4 - 2026-04-06

### Features

- Improve setup and pairing flow with clearer waiting guidance, in-app status feedback, and a reset option for temporary session state without losing completed progress.
- Improve clipboard image and file preview reliability across desktop and web scenarios.
- Improve real-time connection resilience and browser-based real-time session support for web deployments.
- Add telemetry controls with a first-run notice, so you can explicitly choose whether telemetry is enabled.
- Update settings to show sync frequency as coming soon.
- Improve CLI `setup`, `pair`, `connect`, and `reset` commands by handling local daemon lifecycle automatically for more predictable runs.

### Fixes

- Fix a black screen that could appear after unlocking.
- Respect the Auto Unlock preference when the app starts.
- Improve startup readiness and packaging reliability to reduce cases where the app waits too long to become ready.
- Fix clipboard sync edge cases that could cause duplicate, out-of-order, or repeated restore updates.
- Fix settings sync mismatches that could cause inconsistent device configuration.
- Fix quick panel consistency after daemon restarts, including lock state, theme sync across windows, and Windows paste target behavior.
- Restore macOS file previews.
- Fix setup and pairing edge cases that could interrupt active sessions or mis-handle passphrase verification.
- Preserve inline image previews more reliably.
