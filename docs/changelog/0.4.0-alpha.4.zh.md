## 0.4.0-alpha.4 - 2026-04-06

### Features

- 改进初始化与配对流程，提供更清晰的等待提示、应用内状态反馈，以及可重置临时会话状态且不丢失已完成进度的能力。
- 提升剪贴板图片与文件预览在桌面端和 Web 场景下的稳定性。
- 提升实时连接韧性，并在 Web 部署场景下支持基于浏览器的实时会话。
- 新增遥测开关与首次提示，便于你明确选择是否启用遥测。
- 在设置中将同步频率标记为即将推出。
- 改进 CLI `setup`、`pair`、`connect`、`reset` 命令，自动处理本地守护进程生命周期，执行过程更稳定可预期。

### Fixes

- 修复解锁后可能出现黑屏的问题。
- 修复应用启动时未正确遵循自动解锁设置的问题。
- 提升启动就绪与打包可靠性，减少应用长时间未就绪的情况。
- 修复剪贴板同步边界场景，减少重复、顺序异常或反复恢复的问题。
- 修复设置同步不一致导致的设备配置异常。
- 修复守护进程重启后快速面板状态不一致的问题，包括锁定状态、跨窗口主题同步，以及 Windows 粘贴目标行为。
- 修复 macOS 文件预览不可用的问题。
- 修复初始化与配对中的边界问题，避免活跃会话被意外中断或口令校验流程处理异常。
- 提升内联图片预览的稳定性。
