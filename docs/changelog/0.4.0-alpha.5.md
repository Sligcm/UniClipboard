## 0.4.0-alpha.5 - 2026-04-07

### Features

- Improve setup and pairing with clearer status guidance, better waiting feedback, and a reset option for temporary setup state without losing completed progress.
- Improve clipboard image and file preview reliability across desktop and web environments.
- Improve real-time connection stability and browser-based real-time session support in web deployments.
- Add telemetry controls with a first-run notice so you can choose whether telemetry is enabled.
- Show sync frequency in settings as a coming-soon option.
- Improve CLI `setup`, `pair`, `connect`, and `reset` commands by handling local daemon lifecycle automatically.

### Fixes

- Fix a black screen that could appear after unlocking.
- Respect the Auto Unlock preference on app startup.
- Improve startup readiness to reduce cases where the app takes too long to become usable.
- Fix clipboard sync edge cases that could cause duplicate, out-of-order, or repeated restore updates.
- Fix settings sync mismatches that could lead to inconsistent device configuration.
- Fix quick panel state issues after daemon restarts, including lock state, theme synchronization across windows, and Windows paste target behavior.
- Restore macOS file previews.
- Fix setup and pairing edge cases that could interrupt active sessions or mis-handle passphrase verification.
- Preserve inline image previews more reliably.
- Prevent rendering lag when very large text content appears in previews.
