## 0.4.0-alpha.5 - 2026-04-07

### Features

- 改进设置与配对流程，提供更清晰的状态提示和等待反馈，并支持重置临时设置状态且不丢失已完成进度。
- 提升剪贴板图片与文件预览在桌面端和 Web 端的可靠性。
- 提升实时连接稳定性，并改进 Web 部署中的浏览器实时会话支持。
- 新增遥测开关与首次提示，可由你自行选择是否启用遥测。
- 在设置中将同步频率标记为即将推出。
- 改进 CLI 的 `setup`、`pair`、`connect`、`reset` 命令，自动处理本地守护进程生命周期。

### Fixes

- 修复解锁后可能出现黑屏的问题。
- 应用启动时正确遵循自动解锁偏好设置。
- 提升启动就绪稳定性，减少应用长时间不可用的情况。
- 修复剪贴板同步边界问题，避免重复、乱序或重复恢复更新。
- 修复设置同步不一致导致的设备配置异常。
- 修复守护进程重启后的快速面板状态问题，包括锁定状态、跨窗口主题同步以及 Windows 粘贴目标行为。
- 恢复 macOS 文件预览。
- 修复设置与配对流程中的边界问题，避免中断进行中的会话或错误处理口令验证。
- 提升内联图片预览保留的可靠性。
- 防止预览超大文本内容时出现渲染卡顿。
