## 0.4.0-alpha.6 - 2026-04-08

### Features

- Add zoom in and zoom out shortcuts, with interface scale that persists and stays consistent across the main window and quick panel.
- Add file-list preview support so copied files show readable filenames in the quick panel.
- Improve cross-device connection behavior to prefer faster local routes first and reduce failed sync attempts.

### Fixes

- Fix pairing flows that could get stuck after a finished or failed session and block new pairing requests.
- Fix Windows startup cases where global shortcuts could fail to register and prevent normal launch.
- Fix browser-based delete actions so clipboard items can be removed reliably in web deployments.
- Reduce lag when previewing very large text content.
