## 0.4.0-alpha.8 - 2026-04-09

### Fixes

- You can retry installing an app update after a failed download instead of getting stuck.
- Settings now shows a clear "Checking..." state while update detection is running.
- Keyboard zoom shortcuts now follow the same zoom levels as the Settings scale controls, so shortcut zoom stays in sync with the selected scale.
- Fixed a sync issue where copying new content right after receiving synced clipboard content could be skipped.
