## 0.4.0 - 2026-04-09

### Features

- Improve peer connection reliability and responsiveness by preferring better network paths and avoiding immediate retries to recently failed addresses.
- Add file-list previews in clipboard history and the quick panel so copied files are easier to identify before pasting.
- Improve setup and pairing feedback with clearer progress hints and better event delivery during device onboarding.
- Add telemetry controls in settings with a first-run notice so you can choose whether telemetry is enabled.

### Fixes

- Fix large text previews that could freeze or lag the clipboard list and quick panel.
- Fix Windows startup failures caused by stale global hotkey registrations after an unclean previous exit.
- Fix quick panel behavior regressions including paste targeting on Windows, lock-state recovery after daemon restart, and cross-window theme synchronization.
- Fix clipboard restore and sync edge cases that could trigger repeated or out-of-order updates.
- Restore and stabilize file and inline image previews across platforms, including macOS file previews.
