## 0.4.0 - 2026-04-09

### Features

- 提升设备连接的稳定性与响应速度，优先使用更合适的网络路径，并避免立即重试刚失败的地址。
- 在剪贴板历史和快速面板中新增文件列表预览，复制文件后可在粘贴前更容易确认内容。
- 改进设置与配对阶段的反馈，提供更清晰的进度提示，并提升设备接入过程中的事件到达稳定性。
- 在设置中新增遥测开关和首次提示，可由你自行选择是否启用遥测。

### Fixes

- 修复超大文本预览导致剪贴板列表或快速面板卡顿、冻结的问题。
- 修复 Windows 在上次异常退出后因残留全局快捷键注册导致的启动失败问题。
- 修复快速面板行为问题，包括 Windows 粘贴目标异常、守护进程重启后的锁定状态恢复，以及跨窗口主题不同步。
- 修复剪贴板恢复与同步的边界问题，避免重复或乱序更新。
- 恢复并提升跨平台文件与内联图片预览稳定性，包括 macOS 文件预览。
