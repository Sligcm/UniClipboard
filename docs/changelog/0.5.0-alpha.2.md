## 0.5.0-alpha.2 - 2026-04-10

### Features

- Added keyboard shortcuts to zoom the interface, with scale kept in sync between the main window and quick panel.
- Introduced a borderless main window style on Windows with updated frame and panel visuals.

### Fixes

- Quick panel now opens on the screen where your cursor is.
- Fixed quick panel preview behavior so hover and reopen actions no longer cause focus jumps or flash-and-shrink artifacts.
- Improved quick panel resize behavior on Windows so history and preview panes expand more smoothly and stay balanced.
- Matched quick panel preview scrollbar styling with the history list for a more consistent look.
- Improved clipboard preview loading so browsing items in the main window and quick panel feels smoother.
- Restoring file items from clipboard history now works again.
- Tooltips now close reliably when the window loses focus or the pointer leaves quickly.
- Zoom keyboard shortcuts now follow the same scale steps as Settings so zoom controls stay consistent.
- You can now retry updates after a failed download, and update checks show a clear in-progress state.
- Fixed an issue where some local clipboard changes could be misclassified and skipped from sync after a recent remote update.
- Fixed pairing flow issues that could make new requests appear busy after a previous session had already ended.
- Fixed clipboard item deletion failures caused by blocked DELETE requests.
