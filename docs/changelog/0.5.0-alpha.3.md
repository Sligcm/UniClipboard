## 0.5.0-alpha.3 - 2026-04-10

### Features

- Added keyboard shortcuts to zoom the interface, with scale kept in sync between the main window and quick panel.
- Introduced a borderless main window style on Windows with refreshed frame and panel visuals.

### Fixes

- Quick panel now opens on the screen where your cursor is.
- Fixed quick panel preview behavior so hover and reopen actions no longer cause focus jumps, stale flashes, or sudden shrink effects.
- Improved quick panel resize behavior on Windows so history and preview panes expand more smoothly and stay balanced.
- Matched quick panel preview scrollbar styling with the history list for a more consistent look.
- Improved clipboard preview loading so browsing items in the main window and quick panel feels smoother.
- Restoring file items from clipboard history now works again.
- Tooltips now close reliably when the window loses focus or the pointer leaves quickly.
- You can retry app updates after a failed download, and update checks now show a clear in-progress state.
- Fixed a sync issue where some new local clipboard changes could be skipped right after remote updates.
- Fixed pairing and connection flows that could stay busy after a previous session or concurrent connection attempt.
- Fixed clipboard item deletion failures in web deployments.
