## 0.5.0-alpha.3 - 2026-04-10

### Features

- 新增界面缩放快捷键，主窗口与快速面板的缩放比例会保持同步。
- Windows 主窗口改为无边框样式，并更新了窗口框架与面板视觉效果。

### Fixes

- 快速面板现在会在鼠标所在的屏幕上打开。
- 修复快速面板预览行为，悬停和重新打开时不再出现焦点跳动、旧内容闪现或突然收缩的问题。
- 优化 Windows 上快速面板的尺寸变化表现，历史列表与预览面板扩展更平滑且比例更均衡。
- 统一快速面板预览区与历史列表的滚动条样式，视觉更一致。
- 优化剪贴板预览加载方式，主窗口和快速面板浏览条目时更流畅。
- 修复从剪贴板历史恢复文件条目失败的问题。
- 修复窗口失焦或鼠标快速移出时提示浮层可能残留的问题。
- 修复更新下载失败后无法重试的问题，并在检查更新时显示明确的进行中状态。
- 修复最近接收远端更新后，部分新的本地剪贴板变化可能被跳过同步的问题。
- 修复配对与连接流程在上一次会话结束后或并发连接时仍可能显示忙碌的问题。
- 修复 Web 部署中剪贴板条目删除失败的问题。
