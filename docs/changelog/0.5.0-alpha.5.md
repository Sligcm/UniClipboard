## 0.5.0-alpha.5 - 2026-04-12

### Features

- Added local encrypted clipboard search with advanced filters in both the main app and Quick Panel.
- Added search index status and a manual rebuild action in Storage settings.
- Redesigned the clipboard preview panel with clearer metadata and improved file, image, text, code, and link previews.
- Improved file transfer progress display with clearer status, speed, and remaining time details.

### Fixes

- Restored the native window title bar on Windows.
- Improved paired device sync stability by recovering interrupted connections more reliably.
