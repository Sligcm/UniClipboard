## 0.5.0-alpha.6 - 2026-04-12

### Features

- Added local full-text search for clipboard history with search result counts and clearer empty and loading states.
- Added advanced search in the quick panel, including filter tokens and time-range filtering for faster narrowing.
- Added a Search index section in Settings so you can check index status, see last rebuild time, and manually rebuild.
- Redesigned the clipboard preview area with clearer information for text, links, images, and files.
- Improved file transfer progress display with transfer speed and estimated remaining time.
- Improved paired-device connection recovery after sleep, network changes, and other transient disconnects.

### Fixes

- Fixed the Windows app title bar so it shows correctly again.
- Fixed advanced search input ref handling so focus and keyboard behavior work reliably.
- Fixed the pairing PIN dialog so clicking outside no longer closes it unexpectedly.
- Fixed transfer progress matching so active progress is shown more reliably on the correct clipboard item.
