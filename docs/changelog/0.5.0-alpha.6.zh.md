## 0.5.0-alpha.6 - 2026-04-12

### Features

- 新增本地剪贴板历史全文搜索，并提供更清晰的搜索结果数量、空状态与加载状态提示。
- 新增快速面板高级搜索，支持过滤 token 和时间范围筛选，更快缩小结果范围。
- 在设置中新增“搜索索引”区域，可查看索引状态、最近重建时间，并手动触发重建。
- 重新设计剪贴板预览区域，文本、链接、图片和文件的信息展示更清晰。
- 优化文件传输进度展示，新增传输速度与预计剩余时间。
- 提升设备配对后的连接恢复能力，在睡眠唤醒、网络变化和短暂断连后更稳定。

### Fixes

- 修复 Windows 应用标题栏显示异常问题。
- 修复高级搜索输入框的 ref 兼容问题，焦点和键盘行为更稳定。
- 修复配对 PIN 弹窗在点击外部区域时被意外关闭的问题。
- 修复传输进度与条目匹配不稳定的问题，进度更可靠地显示在对应剪贴板条目上。
