## 0.6.0 - 2026-05-02

### Breaking Changes

- 这一版重做了设备之间的联网方式，从 0.5.0 升级后，原有的设备配对会失效。所有设备都需要重新配对一次，之后即可继续同步。

### Features

- 更稳的跨网同步：在不同 Wi-Fi、家庭和办公室网络之间，设备连接成功率明显提升；网络较差或临时抖动时，恢复也更快。
- 休眠 / 切网后秒同步：电脑睡眠唤醒、切换 Wi-Fi 或断网恢复后，无需重新握手即可继续同步，不会再出现"复制后等很久才到"的情况。
- 更清晰的首次设置：首次启动时，可以创建一个新的"个人空间"，也可以通过邀请码 + 口令加入已有空间。
- 设备页升级：在"设备"页可以一键生成短期邀请码，邀请新设备加入；也可以复制、重新生成邀请码，并实时看到对方加入成功或失败。
- 加入其他空间不丢历史：从"设备"页可以加入其他空间，本地的剪贴板历史会被保留并迁移过去。
- 改进同一局域网内的设备发现，让配对和传输更可靠。
- 升级提醒：升级到本版本时会有专门的提示，告诉你需要重新配对，不会再静默失败。
- 命令行（CLI）增强：新增 `uniclip init` 创建空间、`uniclip invite` 生成邀请码、`uniclip join` 加入空间、`uniclip members` 查看已配对设备和在线状态、`uniclip send` / `uniclip watch` 直接通过命令行发送或监听剪贴板内容、`uniclip switch-space` 在已配对设备上切换到另一个空间（自动迁移本地历史）、`uniclip upgrade status / ack` 查看与确认升级提示。无需打开 GUI，也能在终端完成完整的设置、配对与同步流程。

### Fixes

- 修复"已配对设备"列表和数量会把本机也算进去的问题。
- 修复粘贴不带分隔符的邀请码时识别不出的问题。
- 修复从其他设备接收剪贴板内容后，本地历史可能出现重复记录的问题。
- 修复在 Windows 或 macOS 上写入系统剪贴板时，文件列表、PNG 图片和富文本内容偶尔丢失格式的问题。
- 修复退出或重启 App 时偶尔卡顿不响应的问题。
- 修复添加设备时，配对完成后成功状态被新邀请覆盖的问题。
- 移除设备后现在会完整清理该设备，避免继续尝试同步，并支持之后重新配对。
- 全新安装完成首次设置后，不再错误显示重新配对提示。
- 避开不可用的虚拟网络地址，降低 VPN 或代理环境下设备连接和传输失败的概率。
- 修复 Windows 上清理缓存可能删除已保存的设置、初始化状态和历史记录的问题。
- 提高文件和图片剪贴板历史的稳定性，避免缓存文件被删除或过期后引发异常。
- 修复后台服务重启后剪贴板同步不会自动恢复的问题。
- 修复已配对设备在线状态显示不准确的问题，并在打开设备页面时更快更新离线状态。
- 修复快捷面板打开后显示空白窗口的问题。
- 整体提升了冷启动、设备短暂离线、以及网络变化场景下的同步可靠性。
