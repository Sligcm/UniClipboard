## 0.7.0-alpha.1 - 2026-05-05

### Breaking Changes

- On macOS, existing paired devices may need to be paired again once after upgrading before clipboard sync resumes.

### Features

- Added LAN-only Mode in Network settings, with a restart notice and an explanation of what can still reach the internet while it is on.
- Added an option for shared VPN users to allow virtual network addresses when both devices can reach each other through the same virtual network.
- Device cards now show whether each paired device is connected directly, relayed, disconnected, or outside the local network when LAN-only Mode is on.
- The system tray now shows whether LAN-only Mode is active.
- Experimental network settings are now marked with an Experimental badge and tooltip.

### Fixes

- Reduced repeated macOS Keychain prompts during startup, setup, and unlock.
- Fixed macOS Dock icon reopening so it restores the main window after the window has been closed to the tray.
- Fixed Network settings save feedback so restart notices and error messages stay in sync with the saved setting.
