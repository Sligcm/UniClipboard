## 0.7.0-alpha.1 - 2026-05-05

### Breaking Changes

- 在 macOS 上，升级后已有配对设备可能需要重新配对一次，之后才能继续同步剪贴板。

### Features

- 在网络设置中新增 LAN-only 模式，并提供重启提示和开启后仍可能联网的说明。
- 为使用同一 VPN 的设备新增“允许虚拟网络地址”选项，便于两端通过同一个虚拟网络连接。
- 设备卡片现在会显示每台已配对设备是直连、中转、未连接，还是在 LAN-only 模式下不在本地网络。
- 系统托盘现在会显示 LAN-only 模式是否已开启。
- 实验性的网络设置现在会显示“实验性”徽章和提示说明。

### Fixes

- 减少 macOS 在启动、首次设置和解锁时反复弹出 Keychain 授权的问题。
- 修复 macOS 上关闭到托盘后点击 Dock 图标无法恢复主窗口的问题。
- 修复网络设置保存后的反馈，使重启提示和错误信息与实际保存状态保持一致。
