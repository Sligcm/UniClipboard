## 0.7.0-alpha.5 - 2026-05-07

### Features

- Add iPhone Shortcut clipboard sync over the local network, with pairing, QR-code setup, and configuration controls on the Devices page
- Add a Linux Snap package so UniClipboard can be installed on more distributions

### Fixes

- Restore rich text and HTML clipboard history entries without falling back to plain text or truncated content
- Improve window spacing and overlay dragging so dialogs, sheets, and pages fit better with the redesigned title bar
