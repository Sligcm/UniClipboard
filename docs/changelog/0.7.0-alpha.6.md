## 0.7.0-alpha.6 - 2026-05-08

### Features

- Mobile Sync setup now shows separate iOS and Android instructions and credentials for compatible mobile clients.
- Paired mobile devices now show their login username and can get a new password without removing and adding the device again.
- The iOS Shortcut install QR code and link now point to the latest compatible shortcut.
- Paired desktops now return to Online sooner when they reconnect, while unreachable devices are checked less aggressively.

### Fixes

- Images sent from mobile clients now arrive as images even when the client does not provide a specific image type.
- Incoming clipboard items no longer leave a stuck receiving placeholder after the content has arrived.
