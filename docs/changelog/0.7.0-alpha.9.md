## 0.7.0-alpha.9 - 2026-05-10

### Features

- Add RPM packages and a Fedora COPR install channel so dnf-based Linux users can install and update UniClipboard through their package manager.

### Fixes

- Make LAN-only mode keep already-paired devices on local-network discovery and direct connections instead of still reaching them through public network services.
- Make search match partial terms inside copied addresses and code-style text, such as localhost, IP addresses, and mixed-case identifiers.
- Make Quick Panel paste on macOS more reliably return focus to the previous app before pasting the selected item.
