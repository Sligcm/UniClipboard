## 0.8.0-alpha.1 - 2026-05-11

### Features

- Linux Wayland desktops now get more reliable clipboard change detection and clipboard sync.
- Files and images sent from mobile devices now show live receiving progress in the dashboard.

### Fixes

- Pasting invitation codes with a hyphen no longer drops the final character.
- Saving automatic history cleanup rules in Storage settings no longer fails.
- Restarting after network or mobile sync setting changes is more reliable and avoids port conflicts.
- Mobile file and image transfers now show their receiving and completed states in the right order.
- Documentation welcome-page quick links no longer open broken pages.
