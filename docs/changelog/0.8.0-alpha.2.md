## 0.8.0-alpha.2 - 2026-05-11

### Features

- Add a one-tap mobile sync setup flow from the device panel, so first-time device pairing no longer requires opening settings or restarting the app.
- Apply mobile sync enable, disable, and port changes immediately without showing a restart prompt.

### Fixes

- Prevent rapid duplicate clipboard updates from being applied more than once.
- Show an error immediately when mobile sync cannot start on the selected port instead of continuing into device setup.
