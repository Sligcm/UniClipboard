## 0.8.0-alpha.3 - 2026-05-11

### Features

- Make mobile companion setup faster by letting first-time device adding enable local mobile sync and continue without restarting the app.
- Apply mobile sync listener setting changes immediately, including port changes, without asking users to restart.
- Improve Linux clipboard support on X11, including large clipboard content and rapid clipboard changes.

### Fixes

- Prevent repeated synced clipboard content from being applied more than once when a device quickly sends the same content again.
- Reduce repeated clipboard write failures when another app temporarily blocks clipboard access on Windows.
- Show mobile sync listener start failures immediately so users are not sent into device pairing when the local listener is unavailable.
