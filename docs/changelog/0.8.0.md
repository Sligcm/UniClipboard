## 0.8.0 - 2026-05-11

### Features

- Mobile companion setup can now start from the device panel and continue without opening settings or restarting the app.
- Mobile sync enable, disable, and port changes now take effect immediately.
- Files and images sent from mobile devices now show clearer receiving progress and completed states.
- Clipboard content uploaded from a phone now syncs to all paired desktop devices, including text, images, and files.
- Linux clipboard support is more reliable on Wayland and X11, including large clipboard content and rapid clipboard changes.

### Fixes

- Pasting invitation codes with a hyphen no longer drops the final character.
- Saving automatic history cleanup rules in Storage settings no longer fails.
- Mobile sync now reports listener start failures immediately instead of continuing into device setup.
- Prevent rapid duplicate clipboard updates from being applied more than once.
- Reduce repeated clipboard write failures when another app temporarily blocks clipboard access on Windows.
- Larger files uploaded from a phone can now complete instead of failing at the previous size limit.
