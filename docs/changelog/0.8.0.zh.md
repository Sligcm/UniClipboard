## 0.8.0 - 2026-05-11

### Features

- 可以直接从设备面板开始设置移动端同步，不再需要先打开设置或重启应用。
- 移动端同步的启用、停用和端口修改会立即生效。
- 移动端发送文件和图片时，接收进度和完成状态显示得更清楚。
- 手机上传的剪贴板内容现在会继续同步到所有已配对的桌面设备，包括文本、图片和文件。
- Linux 下的剪贴板支持更加可靠，覆盖 Wayland 和 X11，包括大内容和快速连续变化。

### Fixes

- 粘贴带连字符的邀请码时，不会再丢失最后一个字符。
- 存储设置中的自动清理历史规则现在可以正常保存。
- 移动端同步监听启动失败时会立即提示，不再继续进入设备添加流程。
- 避免快速重复同步的剪贴板内容被重复写入。
- 降低 Windows 上其他应用临时占用剪贴板时反复写入失败的问题。
- 手机上传较大文件时不再因为之前的大小限制而失败。
