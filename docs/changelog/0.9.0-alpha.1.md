## 0.9.0-alpha.1 - 2026-05-12

### Features

- Add a plain-text paste shortcut in the quick panel: hold Option/Alt when pressing Enter, clicking an item, or using Command/Ctrl plus a number to paste without formatting.
- Expand appearance settings with separate light and dark theme presets, clickable previews, a Follow system toggle, and custom colors for accent, background, foreground, and borders.
