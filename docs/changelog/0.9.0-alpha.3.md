## 0.9.0-alpha.3 - 2026-05-12

### Breaking Changes

- Clipboard sync now requires paired desktop devices to be on this release or later, because older alpha builds may not receive new clipboard items from upgraded devices.

### Features

- Show the active connection address in the paired devices list so it is easier to see how each online device is connected.
- Make Appearance settings clearer by letting Light, Dark, and System themes be selected directly from preview cards.

### Fixes

- Let users unlock with their space passphrase when automatic unlock cannot recover the saved key.
- Improve Linux responsiveness by reducing heavy visual effects that can slow down rendering.
