## 0.9.0-alpha.5 - 2026-05-13

### Features

- Add a separate privacy switch for usage analytics, so diagnostics and product usage statistics can be controlled independently.
- Improve mobile sync compatibility with Android clients, including clipboard history access and image or file uploads.
- Add an Ask AI chat to the documentation site so users can find UniClipboard help more quickly.

### Fixes

- Make device pairing more reliable on Tailscale and other overlay networks.
- Restore the saved interface scale as soon as the app starts, so zoom shortcuts stay in sync after a restart.
- Make the Linux main window layout fit the system title bar more cleanly.
