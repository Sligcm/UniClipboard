## 0.9.0 - 2026-05-14

### Breaking Changes

- Paired desktop devices should be upgraded together because older versions may not receive new clipboard items from devices running 0.9.0.

### Features

- Add a plain-text paste shortcut in the quick panel: hold Option/Alt when pressing Enter, clicking an item, or using Command/Ctrl plus a number to paste without formatting.
- Expand appearance settings with separate light and dark theme presets, direct Light/Dark/System preview selection, and custom colors for accent, background, foreground, and borders.
- Add separate privacy controls for diagnostics and usage analytics.
- Improve Android mobile sync compatibility, including clipboard history access and image or file uploads.
- Show the active connection address in the paired devices list so it is easier to see how each online device is connected.
- Ask for confirmation before switching the update channel to Alpha.

### Fixes

- Let users unlock with their space passphrase when automatic unlock cannot recover the saved key.
- Make device pairing more reliable on Tailscale and other overlay networks.
- Fix a Windows issue where choosing paste as plain text from the quick panel could reopen the panel instead of pasting into the previous app.
- Restore the saved interface scale as soon as the app starts, so zoom shortcuts stay in sync after a restart.
- Make keyboard shortcut changes take effect immediately after saving, including the global shortcut for opening the quick panel.
- Improve Linux responsiveness and layout so the main window and settings page fit the system title bar more cleanly.
- Check for updates on the newly selected channel immediately after changing update channels.
