## 0.9.0 - 2026-05-14

### Breaking Changes

- 已配对的桌面设备应一起升级，因为旧版本可能无法接收运行 0.9.0 的设备发送的新剪贴板内容。

### Features

- 快捷面板支持按住 Option/Alt 以纯文本粘贴，适用于回车、点击条目以及 Command/Ctrl 加数字快捷键。
- 外观设置现在可分别配置浅色和深色主题预设，并支持通过浅色、深色、跟随系统预览直接切换，以及自定义强调色、背景色、前景色和边框色。
- 新增单独的诊断数据与使用情况分析隐私开关。
- 改进 Android 移动同步客户端兼容性，包括剪贴板历史访问以及图片或文件上传。
- 已配对设备列表现在会显示当前连接地址，方便确认在线设备的连接方式。
- 切换更新频道到内测版前会先要求确认。

### Fixes

- 当自动解锁无法恢复已保存密钥时，用户现在可以输入空间口令手动解锁。
- 提升在 Tailscale 和其他覆盖网络上的设备配对可靠性。
- 修复 Windows 上从快捷面板选择纯文本粘贴时可能重新打开面板，而不是粘贴到原应用的问题。
- 应用启动时会立即恢复已保存的界面缩放，重启后的缩放快捷键也会保持同步。
- 键盘快捷键保存后会立即生效，包括用于打开快捷面板的全局快捷键。
- 改进 Linux 响应速度和布局，让主窗口与设置页更自然地适配系统标题栏。
- 更改更新频道后，会立即按新选择的频道检查更新。
